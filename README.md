# 🚦 Smart Traffic Control System

An AI-powered Smart Traffic Control System that uses YOLOv8 and Computer Vision to detect vehicles, analyze traffic density, and dynamically control traffic signals.

---

## 🔍 Features

✔ Real-time vehicle detection using YOLOv8  
✔ Lane-wise traffic analysis  
✔ Traffic density calculation  
✔ Adaptive traffic signal control  
✔ Live dashboard visualization  
✔ Traffic analytics graph  
✔ Upload and analyze custom traffic videos  

---

## 🛠 Tech Stack

- Python
- Flask
- OpenCV
- YOLOv8
- HTML, CSS, JavaScript
- Chart.js

---

## ⚙️ How It Works

1. Traffic video is processed frame by frame  
2. YOLOv8 detects vehicles in each frame  
3. Vehicles are assigned to lanes  
4. Lane-wise vehicle count is calculated  
5. Traffic density is determined  
6. Signal priority is given to the most congested lane  
7. Results are displayed on a real-time dashboard
